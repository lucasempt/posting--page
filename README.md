# 🚀 Posting Page - Dev Connect

Este projeto simula a criação de posts para redes sociais (como Facebook e LinkedIn), realizando uma comunicação real com a API **JSONPlaceholder**.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Design OLED moderno e responsivo)
- **JavaScript ES6+** (Manipulação de DOM e Fetch API)

## ✨ Diferenciais de UX (Experiência do Usuário)
Para tornar o projeto mais próximo da realidade, adicionei:
- **Skeleton Loader:** Feedback visual enquanto os dados são processados pela API.
- **Loading State:** Botão de postagem desabilitado durante o envio para evitar duplicidade.
- **Toast Notifications:** Mensagem flutuante confirmando o sucesso da operação.
- **Layout Travado:** Design consistente que mantém o tamanho do card independente do volume de texto.

## 🚀 Como executar o projeto
1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador (recomenda-se o uso da extensão Live Server no VS Code).
3. Digite um título, um conteúdo e clique em "Postar Agora".
